# Execute Query

Execute a constructed SQL query and synthesize the results into a user-friendly response.

## Steps
1. Execute the constructed SQL query using the BigQuery execution tool available to you.
2. Wait for the query results to be returned.
3. Interpret the results in the context of the original question.
4. Present the answer clearly — do NOT return raw row data. Summarize, format, or explain as appropriate.
5. If the results are empty or unexpected, communicate that transparently rather than fabricating an answer.
